# Episode 19: Perform a live KVM migration

Run a one-second application probe outside the guest, then migrate its running
state and non-shared storage. Review the probe timestamps to calculate the
last-success to first-success gap. Confirm the source is undefined and the
destination owns the persistent domain before ending the test.
