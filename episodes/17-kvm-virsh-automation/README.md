# Episode 17: Automate KVM provisioning with Bash and virsh

Use a reviewed TSV inventory and the Episode 16 cloud-init provisioner to
create a small fleet. Re-running the deployment is safe: existing domains are
reported and left unchanged.

- Release tag: `episode-17-v1`
- Series index: https://inventivehq.com/blog/kvm-virtualization-series

The included inventory uses documentation-only lab addresses and MACs. Supply
your own public SSH key; no private keys, passwords, or tokens belong here.
