# Episode 20: Monitor and troubleshoot KVM performance

Create CPU, disk, and network symptoms separately and for a bounded duration.
For each symptom, compare libvirt counters with host and guest evidence before
changing configuration. The script removes its temporary files and confirms no
synthetic load remains.

- Release tag: `episode-20-v1`
- Series index: https://inventivehq.com/blog/kvm-virtualization-series

The measurements describe this lab at this moment. They are not universal
benchmarks and should not be used to rank storage, networks, or CPU models.
