# Security policy

Do not open a public issue containing credentials, private keys, tokens, public
IP addresses tied to private infrastructure, or unredacted production logs.

Report a security issue through GitHub's private vulnerability reporting for
this repository when available. Otherwise, use the contact method published on
[InventiveHQ.com](https://inventivehq.com).

The examples in this repository are intended for authorized systems and
disposable labs. They are not a substitute for environment-specific review,
backups, access control, or change management.
